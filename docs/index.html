About the Project

A software tool that enables rapid predictions of stellar collision outcomes, including the number of surviving stars and their properties.

MESA Stellar Models

Built on state-of-the-art stellar models generated with MESA, capturing details of stellar structure across a range of masses and ages.

27,000+ SPH Simulations

Trained on the largest grid of 3D smooth particle hydrodynamic (SPH) simulations of stellar collisions using Starsmasher.

Broader Impact

Enables rapid predictions of stellar collision outcomes, which can be included in large N-body models of dense star cluster dynamics.
